# Migrate Menu to MenuV2

Welcome to the migration of Menu into MenuV2, here we will see how to easily migrate to NewComponent component.

## One of the main changes is...

A description of it...

Example:

```tsx
import { Menu } from '@ultraviolet/ui'

const MyComponent = () => (
  <Menu>
      My text
  </Menu />
)
```

Becomes:

```tsx
import { MenuV2 } from '@ultraviolet/ui'

const MyComponent = () => (
  <MenuV2>
      My text
  </MenuV2 />
)
```
