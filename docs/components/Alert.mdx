# Alert

## Variant

Set variant using `variant` prop.

<Playground>
  <Alert variant="success">
    This text is quite long. Lorem ipsum dolor sit amet, consectetur adipisicing
    elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Alert>
  <Alert variant="warning">
    This text is quite long. Lorem ipsum dolor sit amet, consectetur adipisicing
    elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Alert>
  <Alert variant="info">
    This text is quite long. Lorem ipsum dolor sit amet, consectetur adipisicing
    elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Alert>
</Playground>

## Icon

For basic variants, icon is automatically added. A specific one can be set using `icon` prop. All [icons](/docs-components-icon) are supported.

<Playground>
  <Alert icon="lock" variant="success">
    This text is quite long. Lorem ipsum dolor sit amet, consectetur adipisicing
    elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Alert>
</Playground>
