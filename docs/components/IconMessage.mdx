# IconMessage

Set color using `variant` property with the same keywords of _variant_ colors in `src/theming/colors.js` file.
Set icon using `icon` property with the same keywords of @scaleway/ui `Icon` component.
Set texts using an array in `texts` property. Each entry will be placed in a `<Typography as="p">` component.
