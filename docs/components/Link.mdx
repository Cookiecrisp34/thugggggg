# Link

Creates a styled link.

## Variants

Set variant using `variant` property.

<Playground>
  <Box p={2} backgroundColor="gray200">
    <Boxer my={1}>
      {linkVariants.map(variant => (
        <Link key={variant} href="#" variant={variant}>
          {variant}
        </Link>
      ))}
    </Boxer>
  </Box>
</Playground>

## Link Component

By default link is a `a`. You can specify a custom component (like React Router Link) by changing the theme entry `linkComponent`.

## Absolute URLs

Absolute URLs (starting by `http` or `https`) are automatically detected in `to`. If detected, a `<a href="..." />` is used instead of the default `linkComponent` specified in the theme.
